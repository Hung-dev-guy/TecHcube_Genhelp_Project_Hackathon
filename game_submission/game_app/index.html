Condoms: The most effective way to prevent STIs and pregnancy. Use them correctly every time.

Common STIs: Chlamydia, gonorrhea, syphilis, HPV, herpes, HIV/AIDS.

Testing: Get tested regularly if sexually active. Many STIs have no symptoms.

Prevention: Use protection, limit partners, get vaccinated (HPV, Hepatitis B).
